## Almaty 2050

Bright multi-page concept website for a Smart City 2050 project focused on Almaty.

### Stack

- Next.js
- React
- Tailwind CSS

### Pages

- Home
- Vision
- Urban Systems
- Media
- Sources and Prompts

### Run locally

```bash
npm install
npm run dev
```

### Notes

- Media files can be placed in `public/media`
- Presentation notes and prompts live in `docs/presentation-kit.md`
- Vercel deployment steps live in `docs/vercel-deploy.md`
